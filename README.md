# College Portfolio Website

A personal portfolio website to showcase projects, skills, and contact information.

## Live Demo

- Frontend: [GitHub Pages Link](https://brian-bk23.github.io/project-repo/)  
- Backend: Hosted on [Render](https://project-repo-x3if.onrender.com/submit)

## Features

- Responsive design
- Contact form with Node.js backend
- Smooth scrolling and animations
- Easy to maintain and extend

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **Hosting:** Render (Backend), GitHub Pages (Frontend)  
- **CI:** GitHub Actions for testing
